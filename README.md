# Требования:
- Docker (версия 20.10.0 или новее)
-  Docker Compose (версия 1.29.0 или новее)
- 4 ГБ ОЗУ 
- 2 CPU 
- 500 МБ свободного места на диске

---
# Запуск программы
```docker compose up --build -d```

```./gradlew build```

```./gradlew run```

---
# Описание функционала:
- добавление книги
- получение списка книг
- удаление книги
- редактирование книги
